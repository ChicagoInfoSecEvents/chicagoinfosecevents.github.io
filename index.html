Chicago InfoSec Events Calendar

Welcome

The purpose of this calendar is to help promote and minimize event conflicts for all the in-person security events in the Chicagoland area.

Posting Events

If you would like to promote events, you can send a calendar invite to ChicagoInfoSecEvents@gmail.com.

Note: The sending email will be visible on the public calendar.

If you frequently host events (i.e. ISACA, BurbSec, ISSA, CSA, etc.), the calendar organizers will reach out and ask if you want calendar admin access to manage your events. Direct access will be limited to people well-known by one of the organizers, since they could delete events created by other groups.

Event Requirements

Events should include the following:

  • Event organizer (i.e. ISACA, BurbSec, ISSA, CSA, etc.)
  • Event name
  • Date/time
  • Location
  • URL
    • If the event is NOT open to everyone, please state by invitation (or members) only, and do NOT include a URL.
    • Please include the direct URL, and NOT a shortened URL (i.e. no bit.ly, goo.gl, etc. links)
  • No attachments

Eligibility Criteria

Only events that meet the following conditions should be posted:

  • Must be in person
  • Local to Chicago or hosted by an active member of the Chicago community
    • Exceptions will be made for big events that pull numerous attendees from the Chicago area such as RSA, BlackHat, etc.
  • Discount codes appreciated, but not required
